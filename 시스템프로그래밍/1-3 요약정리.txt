유닉스에서의 파일은 데이터를 읽거나 쓸 수 있는 자원, 대상이다.
파일의 사용형식으로는 일괄처리 형식과 대화 형식이 있따.
파일 기술자의 정수값은 개방된 파일을 식별하고 다른 파일 접근 프리미티브에 인수로 전달하는 역할을 한다.

파일 기술자

정의
 운연 체계에서 파일을 사용 할 때 각 파일에 대한 정보를 유지하는 기억 장치의 한 영역 및 정보
 시스템으로부터 할당 받은 정수의 값

특징
 파일 기술자는 음이 아닌 0으로부터 시작해서 연속적으로 번호가 부여됨
 각 프로세스는 사용자 파일 기술자 테이블을 가짐
 모든 프로세스는 일반적으로 한 순간에 0부터 19까지 총 20개까지 열린 파일을 보유 할 수 있음
 파일 기술자의 처음 3개 값은 특수한 의미
 0 -> 표준 입력(stdin), 1- > 표준 출력(stdout), 2- > 표준 에러(stder)
